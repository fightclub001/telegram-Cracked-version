## Telegram ProMax  
重新定义你的 Telegram 日常

Telegram ProMax 是基于 Telegram 官方开源代码深度定制的 Android 客户端，在保留原版速度、安全与纯净体验的基础上，为重度用户解锁一整套高级能力：闪图 PRO、防撤回、防编辑、全局自动翻译、无限置顶、强制下载/截图/录屏、隐身模式等，让每一条消息、每一个细节，都真正由你掌控。

你可以在我们的官网或 GitHub 页面了解与下载最新版本：  
- 官网：<https://telegarmpromax.com/>  
- GitHub：请在本仓库的 Releases 页面下载同步发布的安装包

---

### 我们做了什么？

在 Telegram 官方体验之上，我们针对真实使用场景进行了大量纯客户端级别的功能增强，不触碰服务器、不改数据库、不伪造数据，只在本地帮你“看得更多、记得更清楚、用得更顺手”。

- **闪图 PRO / 破解闪图**  
  - 查看闪图时新增「转发」「手动销毁」按钮，由你决定何时已读、是否销毁。  
  - 破解闪图自动销毁限制，可重复查看，不再被“阅后即焚”束缚。

- **防撤回 / 防编辑**  
  - 对方撤回的消息仍会保留在本地，并清晰标记为「已撤回」。  
  - 自动保存编辑前的消息原文与编辑记录，随时回看对方修改了什么。

- **全局强制翻译**  
  - 一键开启，全局自动翻译所有非中文消息，无需逐条点击，跨语言群聊和频道阅读像刷中文一样自然。

- **无痕转发**  
  - 默认护去原始来源信息，转发时不暴露最初发送者，更好地保护隐私。

- **强制下载 / 强制截图**  
  - 支持下载原本禁止保存的频道与媒体内容到本地。  
  - 支持对受保护内容截图、录屏，由你决定如何保存自己的阅读记录。

- **无限置顶对话**  
  - 解除官方最多 5 个置顶限制，工作群、家庭群、收藏频道统统能置顶，重要会话永远在你视线范围内。

- **隐身模式**  
  - 阅读消息时可不向对方发送已读状态，悄悄在线，避免社交压力。

- **显示 ID / 默认中文**  
  - 在个人资料、群组、频道页面直接显示 ID，一键长按复制，管理和运维更方便。  
  - 首次启动即为简体中文界面，无需额外设置，开箱即用。

### 基于开源 · 保持纯净与透明

Telegram ProMax 完全基于 Telegram 官方开源项目进行二次开发：

- **源代码来自官方开源项目** `https://github.com/DrKLO/Telegram`  
- **所有改动均为客户端本地逻辑调整与界面增强**，不修改服务器逻辑、不接入第三方数据库、不读取或篡改他人数据。  
- **严格遵守 Telegram 官方 API 与协议**，包括正常的已读回执行为、安全指南与开源协议要求。

换句话说，你在使用的，依然是一款遵循 Telegram 原则的客户端，只是在视觉与功能层面，为高阶用户开启了更多“原本可以但默认没有”的能力。
